## 💻 Projeto

O Profile Settings é um projeto do desafio Bora Codar da Rocketseat que consiste em criar uma configuração de perfil em um site.

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- Git e Github
- Figma
